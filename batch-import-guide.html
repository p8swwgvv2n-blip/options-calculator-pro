离线导入 — 批量获取 IV 数据

备选方案 通过本地工具导出 IV 数据为 JSON 文件，再手动导入到 Options Lab 网页。适合不方便运行代理的场景。

与本地代理的区别：本地代理是实时获取数据（推荐），离线导入是定时导出后手动加载。导入的数据在网页不刷新的情况下一直有效，不会被后续查询覆盖。

一键导出 IV（网页版）

无需安装 Python，双击 HTML 文件在浏览器中打开即可导出。

  1. 下载导出工具

    点击下载 一键导出IV.html

  2. 双击用浏览器打开

    下载后双击文件，浏览器会自动打开导出页面。

  3. 点击「开始导出」

    预设 9 只热门股票（AMD / AMZN / GOOGL / UNH / TSLA / AAPL / NVDA / MSFT / META），点击按钮即可自动获取并下载 JSON 文件。

  4. 导入到 Options Lab

    打开 Options Lab 主页面 → 展开「离线导入 IV 数据」→ 选择刚下载的 JSON 文件 → 点击「导入文件」。

注意：此网页版导出工具需要在能访问 Yahoo Finance 的网络环境下使用（海外 IP 或通过代理）。如果你在大陆且无法直接访问 Yahoo，请使用 Python 版并配置代理。

一键导出（Python 版）

双击即可运行的 Python 脚本，无需命令行输入。

  1. 下载脚本

    点击下载 一键导出期权IV.py

  2. 双击运行

    脚本会自动导出预设 9 只股票的期权 IV 数据，生成 iv-data-YYYYMMDD.json 文件。

    如果需要自定义股票列表，在终端运行：

    python3 一键导出期权IV.py
  3. 导入到 Options Lab

    同网页版，将 JSON 文件导入到主页面即可。

定时自动导出

设置定时任务，每天自动导出最新 IV 数据。

  1. 下载所需文件
    文件说明
    定时导出期权IV.py静默模式的导出脚本
    setup-cron-mac.shMac 一键安装定时任务
    setup-cron-win.batWindows 一键安装定时任务
  2. 安装定时任务

    Mac：双击 setup-cron-mac.sh，自动添加每天 9:00 AM 的 crontab 任务。

    Windows：右键 → 以管理员身份运行 setup-cron-win.bat，自动在任务计划程序中创建定时任务。

  3. 导入数据

    定时任务会在指定目录生成 JSON 文件，每天打开 Options Lab 导入即可。

导入方式

在 Options Lab 主页面的「离线导入 IV 数据」区域，支持两种导入方式：

方式操作
文件导入选择导出的 JSON 文件，点击「导入文件」
粘贴导入打开 JSON 文件，复制全部内容粘贴到文本框，点击「导入粘贴」

数据格式

导入的 JSON 格式如下（由导出工具自动生成）：

{
  "version": 1,
  "generatedAt": "2026-05-27T10:30:00",
  "tickers": {
    "AAPL": {
      "symbol": "AAPL",
      "price": 215.32,
      "timestamp": 1748332200,
      "expiries": [
        {
          "date": "2026-05-30",
          "timestamp": 1748577600,
          "calls": [{ "strike": 215, "iv": 28.5, ... }],
          "puts":  [{ "strike": 215, "iv": 29.1, ... }]
        }
      ]
    }
  }
}
数据持久性：导入的数据在网页不刷新的情况下一直有效。切换到已导入的股票时，会自动使用导入的 IV 数据，不会被后续查询覆盖。刷新页面后数据会丢失，需要重新导入。
