本地代理 — 实时获取期权 IV 数据

推荐 通过本地代理服务器直接获取 Yahoo Finance 实时期权链数据，无需手动导入。

工作原理

在你的电脑上运行一个轻量 Python 脚本，它会在 127.0.0.1:8765 启动一个本地服务。Options Lab 网页自动检测并连接这个服务，查询股票时直接获取实时期权链和 IV 数据。

数据全程在本机流转，不经过任何第三方服务器。

重要提示：Yahoo Finance 在中国大陆 IP 已被封锁。如果你的电脑在大陆，需要同时运行 VPN / SS / Clash 等代理软件，让 Python 脚本通过你的代理访问 Yahoo。

快速开始

  1. 下载代理脚本

    从 GitHub 下载 options-proxy-server.py：

    点击下载 options-proxy-server.py
  2. 启动代理（选择适合你的方式）

    如果你有 Clash（默认端口 7890）：

    python3 options-proxy-server.py --proxy socks5://127.0.0.1:7890

    如果你有 SS/SSR（默认端口 1080）：

    python3 options-proxy-server.py --proxy socks5://127.0.0.1:1080

    如果你有 V2Ray（默认端口 10808）：

    python3 options-proxy-server.py --proxy socks5://127.0.0.1:10808

    如果不确定代理地址，用交互模式：

    python3 options-proxy-server.py --ask

    如果你在海外（不需要代理）：

    python3 options-proxy-server.py
  3. 确认连接成功

    脚本启动后会自动打开浏览器，显示 {"status": "ok"} 表示服务正常。

    同时会测试 Yahoo Finance 连接：

    ✓ Yahoo Finance 连接正常！AAPL 当前价格: $215.32

    如果显示 ✗ Yahoo Finance 连接失败，请检查代理软件是否已启动、端口是否正确。

  4. 打开 Options Lab 网页

    保持代理窗口运行，打开 Options Lab 主页面。

    页面顶部会显示 本地代理已连接，查询股票时将自动获取实时期权 IV。

接口说明

接口说明
/?action=status健康检查
/?action=quote&ticker=AAPL实时报价
/?action=history&ticker=AAPL历史K线（100天）
/?action=options&ticker=AAPL期权链（全部到期日）
/?action=options-expiry&ticker=AAPL&expiry_ts=xxx指定到期日的期权链
/?action=full&ticker=AAPL报价 + K线 + 期权链
小贴士：代理启动后保持窗口运行即可，不需要额外操作。关闭终端窗口或按 Ctrl+C 即可停止代理。网页在代理断开时会自动降级为 HV20 估算或使用手动导入数据。
